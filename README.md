# CampusX / 校易通 - 微信小程序项目说明

CampusX 是一个面向大学生的校园资源共享与信息发布微信小程序，当前项目已经从纯前端 Mock MVP 逐步演进为「小程序前端 + 云函数后端」的系统化项目。项目首期聚焦西南大学场景，支持账号登录、个人资料维护、资源发布与浏览、收藏、消息与会话等基础闭环。

## 项目定位

- 面向校园内二手交易、资料共享、技能服务、宿舍用品流转等高频场景
- 以「信息撮合」为主，不涉及支付、担保交易、履约结算等复杂链路
- 前端以原生微信小程序实现，便于后续持续迭代和部署
- 后端通过云函数 / 接口抽象承接用户、商品、收藏、聊天、消息等能力

## 主要功能

- 🏠 **首页**：学校定位、搜索入口、Banner、分类导航、推荐资源
- 🗂️ **分类页**：左侧分类 + 右侧列表，支持综合 / 最新 / 价格排序
- ➕ **发布页**：图片、标题、分类、价格、面交地点、描述，一键发布
- 📄 **详情页**：轮播图、价格、描述、面交信息、发布者信息、收藏、联系入口
- 💬 **消息页**：系统通知 + 私信会话列表
- 💌 **聊天页**：基于会话的消息收发
- 👤 **我的 / 设置**：头像、昵称、性别、简介、学校信息、账号设置、退出登录
- 🔐 **登录**：支持小程序登录态与用户信息同步

当前默认仅保留 5 个核心分类：**数码电子 / 教材书籍 / 考研资料 / 宿舍用品 / 技能服务**。

## 技术特点

- 原生微信小程序目录结构，便于在微信开发者工具中直接运行
- 统一 API 封装，前端通过 `miniprogram/api/index.js` 调用云函数能力
- 个人资料页支持从本地存储和云端同步数据，并可更新头像、昵称、性别和简介
- 资源、收藏、消息、会话等模块已预留后端扩展空间

## 本地运行

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开开发者工具，导入 `miniprogram/` 目录
3. 使用测试号或你自己的 AppID 进行调试
4. 若项目启用了云开发，请先配置云环境及云函数，再编译运行

### TabBar 图标

`app.json` 中引用了 `miniprogram/images/tab-*.png`。如果仓库中未包含这些图标，可以先使用纯文字 TabBar，删除 `iconPath` 和 `selectedIconPath` 即可。
