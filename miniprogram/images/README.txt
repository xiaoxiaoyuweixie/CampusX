放置底部 TabBar 图标，文件命名：

tab-home.png / tab-home-active.png
tab-category.png / tab-category-active.png
tab-publish.png / tab-publish-active.png
tab-message.png / tab-message-active.png
tab-profile.png / tab-profile-active.png

如暂未提供图标，可直接编辑根目录 app.json，删除每个 tabBar 项中的 iconPath / selectedIconPath 字段，开发者工具会以纯文字方式显示 TabBar。
